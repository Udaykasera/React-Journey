# 📝 React To-Do Practice

This is my first React mini-project. After learning the basics of components and the `useState` hook, I am building this To-Do app to solidify my understanding of CRUD (Create, Read, Update, Delete) operations in React.

## 🎯 The Vision

My goal is to move beyond simple static components and create an interactive application where data changes based on user input. I want to understand how "State" drives the User Interface.

## 🚀 Plan of Action

- [ ] **Phase 1:** Setup the UI with basic CSS (Functional Components).
- [ ] **Phase 2:** Use `useState` to capture text from an input field.
- [ ] **Phase 3:** Render a list of tasks dynamically using `.map()`.
- [ ] **Phase 4:** Add functionality to delete tasks.
- [ ] **Phase 5:** Add a "Complete" toggle (strike-through text).
- [ ] **Phase 6 (Bonus):** Persist tasks using `localStorage` so they don't disappear on refresh.

## 🛠️ Tech Stack

- **Framework:** React (Vite)
- **Styling:** CSS3
- **State Management:** React Hooks (`useState`)

## 📈 Learning Journey

I started this project because I wanted to practice:

1. Handling Form Events in React.
2. Managing Arrays in State.
3. Passing data between components.
